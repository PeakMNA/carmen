## State Management

### Page State (Zustand)

```typescript
interface DemandForecastingState {
  // Filters
  selectedLocation: string | null
  selectedCategory: string | null
  forecastPeriod: number
  forecastMethod: 'moving_average' | 'exponential_smoothing' | 'linear_regression' | 'seasonal_decomposition'

  // Data
  forecasts: InventoryForecast[]
  trends: InventoryTrendAnalysis[]
  optimizations: InventoryOptimization[]
  deadStock: DeadStockAnalysis[]

  // UI State
  isLoading: boolean
  selectedItems: string[]
  sortField: string
  sortDirection: 'asc' | 'desc'

  // Actions
  setFilters: (filters: Partial<DemandForecastingState>) => void
  generateForecast: (params: ForecastParams) => Promise<void>
  selectItem: (itemId: string) => void
  clearSelection: () => void
}
```

### Server Actions

| Action | Description | Input | Output |
|--------|-------------|-------|--------|
| `generateForecast` | Generate demand forecast | `{ itemIds, period, method }` | `InventoryForecast[]` |
| `analyzeTrends` | Perform trend analysis | `{ itemIds, period }` | `InventoryTrendAnalysis[]` |
| `getOptimizations` | Get optimization recommendations | `{ locationId?, categoryId? }` | `InventoryOptimization[]` |
| `analyzeDeadStock` | Analyze dead stock | `{ locationId?, categoryId? }` | `DeadStockAnalysis[]` |
| `applyOptimization` | Apply single recommendation | `{ itemId, action }` | `{ success: boolean }` |
| `bulkApplyOptimizations` | Apply multiple recommendations | `{ itemIds[], action }` | `{ applied: number }` |

## Responsive Design

### Breakpoints

| Breakpoint | Layout | Notes |
|------------|--------|-------|
| Mobile (<640px) | Single column | Stacked KPI cards, collapsed tables |
| Tablet (640-1024px) | 2 columns | Side-by-side charts, scrollable tables |
| Desktop (>1024px) | Full layout | 4-column KPI cards, full tables |

### Mobile Adaptations

1. **KPI Cards**: Stack vertically, full width
2. **Charts**: Single chart visible, swipe to navigate
3. **Tables**: Horizontal scroll with fixed first column
4. **Filters**: Collapsible filter panel
5. **Actions**: Bottom sheet instead of inline buttons
