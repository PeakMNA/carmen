## 9. Responsive Behavior

### 9.1 Breakpoints

| Breakpoint | Width | Layout Changes |
|------------|-------|----------------|
| Mobile | < 640px | Single column, stacked cards |
| Tablet | 640-1024px | 2-column grid |
| Desktop | > 1024px | Full layout as designed |

### 9.2 Mobile Adaptations

- KPI cards stack vertically (2x2 grid)
- Charts display full width
- Tables become card-based lists
- Filters collapse into dropdown menu
- Action buttons become icon-only
- Expandable rows open as modal/sheet
